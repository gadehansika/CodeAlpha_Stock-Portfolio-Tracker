# Stock Portfolio Tracker

## Description
A simple Python application that helps users track stock investments and calculate the total value of their portfolio using predefined stock prices.

## Features
- Track stock holdings
- Calculate total portfolio value
- User-friendly input system
- Uses dictionaries for data management

## Technology Used
- Python
- Visual Studio Code

## How to Run
1. Open the project in VS Code or any Python IDE.
2. Run the `CodeAlpha_Stock Portfolio Tracker.py` file.
3. Enter stock details when prompted.
4. View the calculated portfolio value.
